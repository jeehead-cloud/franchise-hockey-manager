# Team Manager UI Kit

Recreation of Atlas's core hockey-manager app: team dashboard, league standings, playoff bracket, and player profile — navigable via a left sidebar (click-through, single-page demo).

No real league/roster data was provided — team names, players, and stats are invented placeholders. Team color theming (`--team-*`) demonstrates the ice-blue default; swap for a real club palette per `guidelines/colors-team.html`.

Icons: [Lucide](https://lucide.dev) via CDN.
